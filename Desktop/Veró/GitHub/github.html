¿Qué es GITHUB?

Es una plataforma de desarrollo colaborativo y de alojamiento de código fuente que utiliza el sistema de control de versiones Git. Permite a los desarrolladores almacenar, gestionar, y compartir sus proyectos de software. Además de ser una plataforma de alojamiento, GitHub ofrece una variedad de herramientas y funcionalidades para facilitar la colaboración y la gestión de proyectos.
