¿Qué es GIT?

Es una herramienta poderosa y flexible que se ha convertido en el estándar para el control de versiones en el desarrollo de software. Es un sistema de control de versiones distribuido que permite a múltiples desarrolladores trabajar en un proyecto de software simultáneamente sin interferir en el trabajo de los demás. Fue creado por Linus Torvalds en 2005 para gestionar el desarrollo del kernel de Linux.
